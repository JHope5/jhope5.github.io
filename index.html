Welcome to Dr. Wolf's Cyber Escape Room!

You have volunteered to take part in an experiment to be taken into Cyber Space to test a new 'Escape Room' style game for Dr. Wolfeschlegelsteinhausenbergerdorff... Dr. Wolf for short.

1. Edit the code on the left hand side, and use the Execute Code button at the top to make your changes, you can also use the Reset Level button to undo everything.
2. There is some information in the comments in the code, it might help you out...
3. Click on the game map and use the arrow keys or W, A, S & D to move the 'P' (that's you) to the green 'O' (the goal).

Click anywhere to begin!
